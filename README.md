# Ops Tool

This is a tool I built to put all my commonly used terminal commands, repeatable tasks into one place. It also expands my workflows by adding automations that make those tasks quicker to do.

## Description

This started as a vanity project just to build a standalone tool that had my most commonly used commands for the web extension I work on, but now includes a few automations for common workday tasks to try to save a little bit of time. Those tasks include task reporting, FAQs, and Canvas test course resetting.

## Dependencies

* Bun, Electron, TypeScript, PicoCSS, Nodemon, Prettier
